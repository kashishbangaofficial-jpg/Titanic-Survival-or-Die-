# 🚢 Titanic Survival Prediction Dashboard

## 📌 Project Overview

The Titanic Survival Dashboard is an interactive Machine Learning and Data Visualization project built using Python and Streamlit. This application analyzes the famous Titanic dataset and predicts passenger survival using multiple Machine Learning algorithms. It also provides interactive charts, clustering analysis, PCA visualization, and user-based survival prediction.

The project combines supervised and unsupervised learning techniques with an attractive dashboard interface to help users understand data patterns and model performance visually.

---

## 🚀 Features

* Interactive Streamlit dashboard
* Titanic dataset analysis and visualization
* Data cleaning and preprocessing
* Multiple Machine Learning models:

  * Logistic Regression
  * Decision Tree Classifier
  * Linear Regression
  * KMeans Clustering
  * PCA (Principal Component Analysis)
* Passenger survival prediction system
* Interactive filters and charts
* Model accuracy comparison
* Feature importance visualization
* Clustering and dimensionality reduction visualization

## 🎯 Prediction System

Users can enter passenger details such as:

* Passenger class
* Gender
* Age
* Fare
* Family members
* Embarked location

The system predicts whether the passenger would survive based on the selected Machine Learning model.
